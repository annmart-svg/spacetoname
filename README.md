# Цей скрипт - це творчий експеримент із текстом. Програма бере передмову до книги Оскара Уайльда і замінює кожен пробіл між словами на підпис " Anna Martyn " .

Це перетворює класичний твір на унікальний потік слів. Код написаний на JavaScript і використовує прості команди для розділення та об'єднання тексту (split та join).
